About Amock

What A-mock Is

A-Mock is a free web based content-mocking platform. A-Mock can be used to Mock REST/SOAP API service response or any other kind of digital content.

What A-Mock Isn't

Amock is not meant to replace an actual backend. It is also not a website hosting platform.

Where To Use AMOCK

Amock is mainly used on development and test environments. It is not meant for use on production/live environments. It can also used for mocking up content for demonstration purposes.

How to Use A-Mock

It's easy. Just...

  • Go to the website https://amock.io
  • Create an account (or you can create your mock as an annonymous user in which case the mock expires after 24 hours)
  • Create Your Mock
  • Access your mock with the mock endpoint you created.
